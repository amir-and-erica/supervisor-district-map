## To run locally
Clone into onto your local machine
```
git clone https://github.com/amir-and-erica/supervisor-district-map
```
Navigate to directory
```
cd supervisor-district-map
```

Install dependencies
```
npm install
```

Run webpack hotload server
```npm run start```
